# Movie-Guide-App
A frontend application enabling users to view the different details  about any movie/series they enter.
It was developed using HTML5, CSS# and Vanilla JavaScript.
This app incorporates the OMDB API to fetch the movie/series details whose name user enters such as the Title, Poster, Runtime, Year of release, IMDB Rating, Appropriate viewing age, Plot, Cast and Genre.
This application make use of concepts like responsive web design, DOM manipulation and asynchronous methods, to provide a seamless and user-frindly UI-UX to the user.
